HTML

HTML es el lenguaje de marcado predominante para la elaboración de páginas web.

Qué es HTML

Significa HyperText Markup Language y define la estructura básica de un sitio.

Historia de HTML

Fue creado por Tim Berners-Lee en 1991 para compartir documentos científicos.

HTML5

Es la quinta revisión importante del lenguaje, que añade etiquetas semánticas y multimedia.
